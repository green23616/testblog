## Key Decisions

### Architecture
- **App Router over Pages Router**: Better for RSC and modern Next.js patterns
- **Server Actions over API Routes**: Simpler data mutations, better TypeScript support
- **Supabase over custom backend**: Faster development, managed infrastructure

### Styling
- **TailwindCSS**: Utility-first, consistent design system
- **Shadcn/ui**: Pre-built accessible components
- **Pretendard Font**: Korean language support with clean aesthetics

### State Management
- **React Server Components**: Default for data fetching
- **Client Components**: Only for interactive features
- **No global state library**: Keep it simple initially
